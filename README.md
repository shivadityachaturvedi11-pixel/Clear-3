# House Price Prediction - Model Validation & Hyperparameter Tuning

This repository contains **Task-3** of the Machine Learning Internship project focused on enhancing a House Price Prediction system using the **California Housing Dataset**. 

The goal of this phase is to move beyond simple baseline tracking to implement robust model validation, diagnose overfitting, and optimize model performance using cross-validation and automated hyperparameter tuning.

## 📁 Repository Structure
* `AI_ML_Task3_Model_Validation_Tuning.ipynb`: Complete Jupyter Notebook containing data preprocessing, model exploration, overfitting diagnostics, and hyperparameter tuning.
* `Model_Validation_Report.md`: Complete documentation detailing overfitting analysis, tuning methodology, and final model justification.

## 🛠️ Tools & Technologies Used
* **Language:** Python 3
* **Environment:** Jupyter Notebook
* **Libraries:** `pandas`, `numpy`, `scikit-learn`, `matplotlib`
